## Summary

T-3.5 implements approved amendment DCR-5 (Option a): the strict `TaskTraceability` gate stays
strict (its regexes are unchanged), and the burden of conformance moves onto the greenfield playbook
prompt. The fix is a production-prompt change plus regression tests. In `GreenfieldPlaybook` the
REQUIREMENTS phase block now directs the model to author each user story as `US-<n>`, each acceptance
criterion as `AC-<n>.<m>`, and each NFR as `NFR-<NAME>` (the gate-recognizable requirement-symbol
shapes, AC-2.5), and the TASKS phase block now directs each task to carry a stable id of the form
`T-<n>` / `T-<n>.<m>` (hyphen mandatory, AC-2.2) and to cite >= 1 of those requirement symbols on its
line. The class + method Javadoc record the DCR-5 contract. Regression tests pin the exact prior
live-failing forms — hyphen-less `T1`/`T2`/`T10` ids citing bare `R1`–`R6` refs are correctly refused
by the strict gate (0 tasks recognized -> not traceable), and the gate-vocabulary forms pass — the
class of defect the existing `TaskTraceabilityTest` could not catch because all ten of its prior tests
used gate-conformant forms only. `TaskTraceability.java` was not modified.

### Praise

**[P1] The gate stays strict; the prompt carries the burden — DCR-5 Option a honoured exactly.**
`TaskTraceability.java` is byte-for-byte unchanged (verified by Read against the inlined authoritative
copy): no `TASK_LINE` widening to accept hyphen-less `T1`, no `REQUIREMENT_REF` widening to accept bare
`R<n>`. The fix is entirely a prompt-emission change plus regression tests, which is precisely the
rejected-Option-b / accepted-Option-a split the ADR pins. The formal `US -> AC -> NFR/ADR -> task`
traceability guarantee is preserved.

**[P2] The regression tests pin the exact named live-failing forms, not a generic substitute.** The
new `priorLiveFailingBreakdownIsRefused` feeds the literal prior live shape (`T1`/`T2`/`T10` ids citing
`R1`–`R6`), asserts 0 tasks recognized and not-traceable, and pairs with `gateVocabularyBreakdownPasses`
for the positive counterpart. This is the precise class of defect the prior 10-test suite could not
catch (all prior inputs used gate-conformant `T-<n>` ids + `AC-/US-/NFR-` refs), so the regression can
never silently reappear.
