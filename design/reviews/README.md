# Reviews

The approval artifacts for the design baseline. **"Silence made explicit"** — a review file records that an artifact was reviewed and approved (or what changed) against a specific commit. No review file → not approved.

- One file per reviewed sub-phase or artifact: `YYYY-MM-DD-<slug>-r<N>.md`, where `N` increments per review round.
- Use `TEMPLATE.md` as the starting point.
- Phase 5 **code** reviews live under `reviews/code/` (written by the reviewer subagent), separate from these design reviews.
